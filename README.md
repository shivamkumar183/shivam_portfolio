# Shivam Kumar — Portfolio

This is a modern, dark-themed responsive portfolio built with React, Vite, Tailwind CSS, Framer Motion and Swiper.

Quick start:

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

Notes:
- Replace EmailJS credentials in `src/components/Contact.jsx` with your own.
- Add actual profile image and links in the components.
